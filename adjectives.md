# Adjectives

Adjectives are 4 or 5 character long and are ending usually with:

* vowels - {"o", "e", "i"} never {"a" or "u"}
* semivowels - {"l","j","y"}
* adjectives ending in consonant can be 6 character long


# Fundamental adjectives

* bono = good  (benevolent)
* malo = bad   (malevolent)
* vero = the truth
* faso = the lie   
* rito = right
* rogo = wrong
* nito = clean
* salo = dirty
