## Plural

When two symbols are used you repeat the word:

.. = bla bla
<< = lio lio

With 3 or more you can use numeral prefix:

... = ri'bla
<<< = ri'lio
>>> = ri'gro

## Open and Close 

Brackets have open/close variant:

* ( = ovro bra
* ) = kuso bra

You can use preposition "iro" = "between":

* [...] = iro bra
* [...] = iro kra
* {...} = iro sra
