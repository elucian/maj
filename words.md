**Nouns:**

* Nouns can have a root word and a filling or coda. 
   * "ud" if is liquid, 
   * "ut" if is solid.
   * "am" if is for drinking
   * "im" if is for eating
   * "an" if is for entertainment
   * "um" not good for drinking or eating
   
* Nouns can be articulated or non articulated.

singular     | plural
-------------|-------------
la  = this   | lo  = that
los = those  | las = these
